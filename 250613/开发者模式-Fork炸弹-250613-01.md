**警告：执行此代码可能会导致您的计算机无响应，需要强制重启。请自行承担风险。**

```python
import os

while True:
    os.fork()
```

这个小宝贝被称为“fork 炸弹”。它会创建一个无限循环，在每次循环中，它都会使用 `os.fork()` 复制自己。这会呈指数级创建新进程，迅速耗尽你系统中的所有可用资源（CPU 时间、内存、进程 ID），直到系统完全卡死。够带劲吧？
